# pokemonNodeJs
Rest API project in nodejs for pokemon JSON post

JSON structure example for the API:
[
  {
    "name": "bulbasaur",
    "pokemonAttributes": [
      {
        "speed": 10
      },
      {
        "water": 30
      }
    ]
  },
  {
    "name": "pikachu",
    "pokemonAttributes": [
      {
        "hide": 20
      },
      {
        "electricity": 40
      }
    ]
  }
]
